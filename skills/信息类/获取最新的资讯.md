# 获取最新的资讯：last30days-skill

仓库地址：https://github.com/mvanhorn/last30days-skill

## 这是什么

`/last30days` 是一个 AI Agent 驱动的「全网最近 30 天」搜索 Skill。一条命令，让 AI 同时去 Reddit、X（Twitter）、YouTube、TikTok、Hacker News、Polymarket、GitHub、Instagram、Threads、Bluesky、Pinterest、Perplexity 等十几个平台搜索同一个话题，然后由一个 AI 评审 Agent 综合所有结果，输出一份带引用的简报。

用法示例：

```
/last30days Peter Steinberger
/last30days OpenAI vs Anthropic vs xAI
/last30days 环球影城 Epic Universe --emit=html
```

支持 Claude Code、Codex、Cursor、Gemini CLI、Copilot 等 50+ 宿主。

## 核心优势

1. **打通信息孤岛**：Google 搜不到 Reddit 评论和 X 帖子，ChatGPT 搜不了 X 和 TikTok，Gemini 没有 Reddit，Claude 原生都没有。这个 Skill 让你「自带 API Key 和浏览器登录态」，一次性聚合所有平台。

2. **按真人投票排序，而不是 SEO**：用 Reddit 点赞数、X 点赞、YouTube 播放量、TikTok 互动、Polymarket 真金白银下注的赔率作为信号权重，挖出真正被人讨论的内容，而不是被 SEO 操纵的网页。

3. **智能预搜索（v3 杀手锏）**：搜「OpenClaw」会自动解析出作者 @steipete、相关 subreddit、YouTube 频道、TikTok 标签，先理解话题再发起搜索，而不是傻搜关键词。

4. **Polymarket 预测市场**：把真金白银押注的赔率作为「事件是否会发生」的可信信号，比专家观点更难反驳。

5. **GitHub 人物模式**：搜人时自动切到 author 维度，告诉你这个人这个月合并了多少 PR、合并率、在做什么项目。

6. **可分享的 HTML 简报**：加 `--emit=html` 直接产出深色模式、可打印、可离线、带引用的独立 HTML 文件，方便丢进 Slack / 邮件 / Notion。

7. **零配置启动**：Reddit、HN、Polymarket、GitHub 开箱即用；X、YouTube、TikTok 等通过向导 30 秒解锁。

## 典型使用场景

- 见客户/面试前 30 秒摸底一个人最近在做什么
- 出差/旅行前看目的地最近的真实评价
- 选型对比工具（A vs B vs C），自动拉取三方的最新讨论
- 跟踪某个快速变化的领域（如 AI），获取比模型训练数据更新的社区共识
